head 요소안에 style요소를 만들고 그 안에 선택자와 규칙을 작성한다. 

        예시 :
    
    <head>
        <style>
            div {
                width: 300px;
                height: 200px;
                color: blue;
            }

            .myclass {
                color: red;
            }
        </style>
    </head>
