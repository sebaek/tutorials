JavaScript HTML DOM

HTML DOM과 함께 JavaScript는 HTML 문서의 모든 요소에 접근하고 변경할 수 있다.

HTML DOM(Document Object Model)

페이지가 로드되면 브라우저는 페이지의 Document Object Model을 만든다.

HTML DOM은 Obejct들의 나무 모양으로 구성된다.

The HTML DOM Tree of Objects

객체 모델과 함께 JavaScript는 동적 HTML을 만드는 모든 능력을 갖는다.

  • 페이지의 모든 HTML 요소를 변경할 수 있다.
  • 페이지의 모든 HTML 속성을 변경할 수 있다.
  • 페이지의 모든 CSS 스타일을 변경할 수 있다.
  • 존재하는 HTML 요소와 속성을 제거할 수 있다.
  • 새로운 HTML 요소와 속성을 만들어 첨가할 수 있다.
  • 페이지에 존재하는 모든 HTML 이벤트에 반응하여 동작할 수 있다.
  • 페이지에 새로운 HTML 이벤트를 만들 수 있다.

앞으로 배울 내용

  1. 요소의 컨텐트를 어떻게 바꿀 것인가.
  2. HTML요소의 스타일(CSS)를 어떻게 바꿀 것인가.
  3. HTML DOM 이벤트를 활용하는 방법
  4. HTML요소를 어떻게 첨가하고 제거할 것인가

HTML DOM이란 무엇인가?

HTML DOM은 HTML을 위한 프로그래밍 인터페이스와 표준 객체 모델이다.

  • 객체로서의 HTML 요소
  • HTML요소의 속성들(properties)
  • 모든 HTML요소에 접근하는 메소드
  • 모든 HTML요소를 위한 이벤트

즉 HTML DOM은 HTML 요소를 어떻게 찾아내고 바꾸고 첨가하고 지우는 방법에 대한 표준이다.

HTML DOM 메소드

HTML DOM 메소드는 우리가 HTML 요소를 제어하는 기능이다.

HTML DOM 속성은 우리가 바꿀 수 있는 HTML 요소의 값이다.

DOM 프로그래밍 인터페이스

DOM에서 모든 HTML요소는 객체로 정의된다.

프로그래밍 인터페이스는 각 객체의 메소드와 속성이다.

프로퍼티는 우리가 읽거나 변경할 수 있는 요소의 값이다.

메소드는 우리가 HTML 요소를 제어할 수 있는 기능이다.

예제

<html>
<body>

<p id="demo"></p>

<script>
document.getElementById("demo").innerHTML = "Hello World!";
</script>

</body>
</html>

HTML 요소 찾기

HTML 요소를 조작하기 위해서는 먼저 그 요소를 찾아야 한다.

찾는 방법들:

  • CSS 셀렉터로 찾기
  • HTML 객체 콜렉션으로 찾기

이벤트 사용하기

HTML DOM은 어떤 이벤트가 발생했을 때 코드를 실행 시킬 수 있게 한다.

이벤트는 HTML 요소에 '어떤일이 발생할 때' 브라우저에 의해서 생성된다:

  • 어떤 요소가 클릭 됐을 때
  • 페이지가 로드 됐을 때
  • input 필드가 변경됐을 때
document.getElementById(id).style.property = new style

이 예제는 사용자가 버튼을 클릭했을 때 id가 'id1'인 요소의 스타일을 바꾸는 예제이다.

<!DOCTYPE html>
<html>
<body>

<h1 id="id1">My Heading 1</h1>

<button type="button" 
onclick="document.getElementById('id1').style.color = 'red'">
Click Me!</button>

</body>
</html>

이벤트에 반응하기

JavaScript는 사용자가 HTML 요소를 클릭하는 등의 이벤트가 발생했을 때 실행될 수 있다.

사용자가 어떤 요소를 클릭했을 때 실행시키기 위해서 JavaScript 코드를 HTML 이벤트 속성에 넣어준다:

onclick=JavaScript

HTML 이벤트의 예:

  • 사용자가 마우스를 클릭했을 때
  • 웹 페이지의 로딩이 완료 됐을 때
  • 이미지의 로딩이 완료 됐을 때
  • 요소위로 마우스가 움직였을 때
  • input 필드가 변경됐을 때
  • HTML 폼이 제출 됐을 때
  • 사용자가 키를 눌렀을 때

아래 예제는 사용자가 h1요소를 클릭했을 때 h1요소의 컨텐트가 바뀌는 예제이다.

<!DOCTYPE html>
<html>
<body>

<h1 onclick="this.innerHTML = 'Ooops!'">Click on this text!</h1>

</body>
</html>

addEventListener() 메소드

이벤트 리스터를 추가하는 예제

document.getElementById("myBtn").addEventListener("click", displayDate);
직접 해보기

addEventListener() 메소드는 특정 요소에 이벤트 핸들러를 첨가한다.

addEventListener() 메소드는 기존에 존재하는 이벤트 핸들러를 덮어쓰지 않고 추가 할 수 있다.

하나의 요소에 여러 이벤트 핸들러를 첨가할 수 있다.

같은 이벤트에 해당하는 핸들러를 여러개 추가할 수 있다.

removeEventListener()로 이벤트 리스너를 제거할 수 있다.

문법

element.addEventListener(event, function, useCapture);

첫번째 파라미터는 이벤트의 타입이다.(click이나 mousedown 같은)

두번째 파라미터는 이벤트가 발생했을 때 실행되어야하는 함수이다.

세번째 파라미터는 boolean값이며 파라미터에 값을 넣거나 넣지 않아도 된다. 이벤트 버블링을 할 지 이벤트 캡쳐링을 할지 결정한다.

이벤트를 입력할 때 'on'을 빼고 입력한다. 즉 'onclick'대신 'click'를 입력한다.
