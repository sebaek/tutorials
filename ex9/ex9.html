<form> 요소

HTML 폼 요소는 사용자에게 값을 입력 받을 때 사용한다.


    <form>
    ...
    form elements
    ...
    </form>

action 속성

action 속성은 폼이 어디로 제출되어야 하는지 정하는 속성이다.

보통 폼의 데이타는 submit 버튼을 클릭하면 서버에 있는 웹페이지로 전송된다.


    <form action="/action_page.php">

method 속성

method 속성은 HTTP method - 전송방식 (GET 또는 POST)를 결정한다.

보통 폼의 데이타는 submit 버튼을 클릭하면 서버에 있는 웹페이지로 전송된다.


    <form action="/action_page.php" method="get">
    

    <form action="/action_page.php" method="post">

GET 전송방식

GET을 사용하면 전송하는 값이 페이지 주소 필드에 보여진다.

민감한 정보를 전송할 때 GET을 사용하지 않는다.

전송하는 양의 제한이 있다.

POST 전송방식

개인정보나 민감한 정보를 전송할 때 항상 POST 방식을 사용한다.

페이지 주소 필드에 보여지지 않는다.

전송되는 데이터 양에 제한이 없다. 파일 전송등에도 사용된다.

<input> 요소

type 속성에 따라 입력하는 방법이 결정된다.

텍스트 입력

<input type="text"> 가 텍스트 입력 필드를 만든다.


    <form>
        First name:<br>
        <input type="text" name="firstname"><br>
        Last name:<br>
        <input type="text" name="lastname">
    </form>

패스워드 입력

<input type="password"> 가 패스워드 입력 필드를 만든다.


    <form>
        User name:<br>
        <input type="text" name="username"><br>
        User password:<br>
        <input type="password" name="psw">
    </form>

제출(submit) 버튼

<input type="submit"> 가 제출 버튼을 만든다.


    <form action="/action_page.php">
        First name:<br>
        <input type="text" name="firstname" value="Mickey"><br>
        Last name:<br>
        <input type="text" name="lastname" value="Mouse"><br><br>
        <input type="submit" value="제출하기">
    </form>

초기화(reset) 버튼

<input type="reset"> 가 초기화 버튼을 만든다.


    <form action="/action_page.php">
        First name:<br>
        <input type="text" name="firstname" value="Mickey"><br>
        Last name:<br>
        <input type="text" name="lastname" value="Mouse"><br><br>
        <input type="submit" value="Submit">
        <input type="reset">
    </form>

라디오 버튼

<input type="radio"> 가 라디오 버튼을 만든다.


    <form>
        <input type="radio" name="gender" value="male" checked> Male<br>
        <input type="radio" name="gender" value="female"> Female<br>
        <input type="radio" name="gender" value="other"> Other
    </form>

체크 박스

<input type="checkbox"> 가 체크박스를 만든다.


    <form>
        <input type="checkbox" name="vehicle1" value="Bike"> I have a bike<br>
        <input type="checkbox" name="vehicle2" value="Car"> I have a car 
    </form>

버튼

<input type="button"> 가 버튼을 만든다.


    <input type="button" onclick="alert('Hello World!')" value="Click Me!">

<select> 요소

<select> 는 드롭다운 리스트를 만든다.


    <select name="cars">
        <option value="volvo">Volvo</option>
        <option value="saab">Saab</option>
        <option value="fiat">Fiat</option>
        <option value="audi">Audi</option>
    </select>

<textarea> 요소

<textarea> 는 여러 줄의 텍스트를 입력하는 요소를 만든다.


    <textarea name="message" rows="10" cols="30">
       The cat was playing in the garden.
    </textarea>

<button> 요소

<button> 는 버튼 요소를 만든다.


    <button type="button" onclick="alert('Hello World!')">Click Me!</button>

input 요소의 속성들

value 속성

value 속성은 input요소의 기본 값을 설정한다.


    <form action="">
    First name:<br>
    <input type="text" name="firstname" value="John">
    </form>

readonly 속성

readonly 속성은 읽기만 가능하도록 한다(값 변경 불가).


    <form action="">
    First name:<br>
    <input type="text" name="firstname" value="John" readonly>
    </form>

disabled 속성

disabled 속성은 요소를 사용하지 못하도록한다. 값이 전송되지 않는다.


    <form action="">
    First name:<br>
    <input type="text" name="firstname" value="John" disabled>
    </form>

size 속성

size 속성은 요소의 크기를 지정한다.


    <form action="">
    First name:<br>
    <input type="text" name="firstname" value="John" size="40">
    </form>

maxlength 속성

maxlength 속성은 요소에 입력하는 값을 제한한다.


    <form action="">
    First name:<br>
    <input type="text" name="firstname" maxlength="10">
    </form>
