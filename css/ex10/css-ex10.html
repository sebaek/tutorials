테두리 꾸미기

border로 시작하는 프로퍼티로 테두리를 꾸밀 수 있다.

테두리 스타일

border-sytle 로 테두리의 모양을 지정한다.

값의 종류들

  • dotted
  • dashed
  • solid
  • couble
  • groove
  • ridge
  • inset
  • outset
  • none
  • hidden
    예시 :
    
    p.dotted {border-style: dotted;}
    p.dashed {border-style: dashed;}
    p.solid {border-style: solid;}
    p.double {border-style: double;}
    p.groove {border-style: groove;}
    p.ridge {border-style: ridge;}
    p.inset {border-style: inset;}
    p.outset {border-style: outset;}
    p.none {border-style: none;}
    p.hidden {border-style: hidden;}
    p.mix {border-style: dotted dashed solid double;}

테두리 폭

border-width 로 테두리의 폭을 지정한다.

    예시 :
    
    p.one {
        border-style: solid;
        border-width: 5px;
    }
    p.three {
        border-style: solid;
        border-width: 2px 10px 4px 20px;
    }

테두리 색

border-color 로 테두리의 색을 지정한다.

    예시 :
    
    p.one {
        border-style: solid;
        border-color: red;
    }

    p.two {
        border-style: solid;
        border-color: green;
    }

테두리 각 면의 스타일 지정

border-top-, border-right-, border-bottom-, border-left- 로 테두리의 각 면의 스타일을 지정한다.

    예시 :
    
    p {
        border-top-style: dotted;
        border-right-style: solid;
        border-bottom-style: dotted;
        border-left-style: solid;
    }

약칭으로 테두리 스타일 지정

border 로 테두리의 폭, 모양, 색을 한번에 지정한다.

    예시 :
    
    p {
        border: 5px solid red;
    }
