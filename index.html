HTML TUTORIAL

개요

  • HTML은 HyperTextMarkupLanguage의 약자이다.
  • HTML은 웹페이지의 구조를 나타내는 텍스트문서이다.
  • HTML 요소(element)들이 HTML 페이지를 구성한다.
  • 요소는 <태그명> 내부 컨텐츠 </태그명> 로 구성됩니다.
  • 각 HTML 태그명은 제목(heading), 문단(paragraph), 테이블(table) 같은 단어를 줄임말이다. 태그 리스트
  • < >- 태그는 앵글브래킷과 태그명으로 구성된다.
  • 웹 브라우저가 HTML문서를 읽고 화면에 보여준다. - 태그는 보여주지 않는다.
  • 종료태그와 내부컨텐츠가 필요없는 요소도 있다.
    • 빈 요소(empty element)라고 한다.
    • <br> - 라인 브레이크
    • <img> - 이미지
  • 문서편집기만 있으면 시작할 수 있다.
