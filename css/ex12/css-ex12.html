패딩

패딩 속성은 컨텐트 주위에 공간을 만들기 위해 사용한다.

패딩 속성은 보더와 콘텐트 사이의 공간의 사이즈를 정한다.

    예시 :
    
    p {
        padding-top: 100px;
        padding-bottom: 100px;
        padding-right: 150px;
        padding-left: 80px;
    }

패딩 속성의 네 개의 값은 top, right, bottom, left 순이다

    예시 :
    
    p {
        padding: 100px 150px 100px 80px;
    }

패딩 속성의 두 개의 값 중 첫 번째는 top, bottom이고 두 번째는 right, left 이다

    예시 :
    
    p {
        padding: 100px 80px;
    }

패딩 속성의 한개의 값은 top, bottom, right, left 를 같은 값으로 정의한다.

    예시 :
    
    p {
        padding: 80px;
    }
