Box Model

모든 HTML요소는 박스로 볼 수 있다. CSS에서 이 것에 대해 박스 모델이라고 한다.

CSS 박스 모델은 모든 HTML 요소가 박스형태의 마진, 보더, 패딩과 컨텐트로 이루어져있다.

아래 박스 모델을 그림으로 나타내었다.

  • 컨텐트를 위해 마련된 공간이 가장 안쪽에 있다.
  • 패딩은 보더와 컨텐트 사이의 간격이다.
  • 보더는 테두리이며 너비를 지정할 수 있다.
  • 마진은 다른 요소와의 간격을 나타낸다.
  • 패딩과 마진은 투명하다
  • 기본 width와 height는 컨텐트 영역의 크기를 나타낸다.
  • box-sizing 속성을 border-box로 설정하면 width와 height의 크기가 border영역까지 포함하여 나타낸다.
