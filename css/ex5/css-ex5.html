의사 클래스 (pseudo-classes)

의사 클래스는 요소의 특정한 상태를 정의 한다.

예를 들어 다음과 같은 경우에 사용한다.

  • 요소 위에 마우스를 올렸을 때 스타일을 적용하고 싶을 때
  • 링크에서 방문했던 요소와 방문하기 전의 요소가 다르게 보이고자 할때
  • 포커스를 얻은 요소에 스타일을 적용할 때
  • 특정 자식 요소만 선택적으로 선택할 때

사용 예시 

    예시 :
    
    selector:pseudo-class {
        property: value;
    }

    /* 방문 전 link */
    a:link {
        color: red;
    }

    /* 방문 후 link */
    a:visited {
        color: green;
    }

    /* 마우스 올렸을 때 link */
    a:hover {
        color: hotpink;
    }

    /* 선택했을 때 link */
    a:active {
        color: blue;
    }
