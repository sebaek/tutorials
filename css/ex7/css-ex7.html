텍스트 색

텍스트 색은 color property로 바꾼다.

    예시 :
    
    body {
        color: blue;
    }

텍스트 정렬

텍스트 정렬은 text-align로 바꾼다.

    예시 :
    
    h1 {
        text-align: center;
    }

    h2 {
        text-align: left;
    }

    h3 {
        text-align: right;
    }

텍스트 데코레이션

텍스트 데코레이션은 text-decoration로 설정한다.

    예시 :
    
    a {
        text-decoration: none;
    }

    h1 {
        text-decoration: overline;
    }

    h2 {
        text-decoration: line-through;
    }

    h3 {
        text-decoration: underline;
    }

폰트

폰트는 font-family로 설정한다.
font-family에 적힌 순서대로 폰트를 적용한다.
아래 예제에서는 Times New Roman 폰트가 없으면 Times폰트를 찾고 없다면 브라우저가 정의한 serif 폰트를 적용한다. 

    예시 :
    
    p {
        font-family: "Times New Roman", Times, serif;
    }

폰트 스타일

폰트 스타일은 font-style로 설정한다.

    예시 :
    
    p {
        font-style: italic;
    }

폰트 사이즈

폰트 사이즈은 font-size로 설정한다.

    예시 :
    
    p {
        font-size: 40px;
    }

폰트 굵기

폰트 굵기는 font-weight로 설정한다.

    예시 :
    
    p {
        font-weight: bold;
    }
