마진

마진 속성은 요소 주위에 공간을 만들기 위해 사용한다.

마진 속성은 보더 바깥의 공간의 사이즈를 정한다.

    예시 :
    
    p {
        margin-top: 100px;
        margin-bottom: 100px;
        margin-right: 150px;
        margin-left: 80px;
    }

마진 속성의 네 개의 값은 top, right, bottom, left 순이다

    예시 :
    
    p {
        margin: 100px 150px 100px 80px;
    }

마진 속성의 두 개의 값 중 첫 번째는 top, bottom이고 두 번째는 right, left 이다

    예시 :
    
    p {
        margin: 100px 80px;
    }

마진 속성의 한개의 값은 top, bottom, right, left 를 같은 값으로 정의한다.

    예시 :
    
    p {
        margin: 80px;
    }
